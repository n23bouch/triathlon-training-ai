# Triathlon Training AI

Projet personnel de génération de plans d'entraînement personnalisés pour triathlon.

## Objectif

Construire un moteur capable de générer des plans d'entraînement adaptés au profil de l'athlète :
- course à pied
- vélo
- natation
- renforcement
- mobilité

## Stack prévue

- Python
- Pydantic
- Pytest
- Ruff
- Black

## Structure du projet

- `src/triathlon_planner/` : code principal
- `tests/` : tests
- `docs/` : documentation produit et technique
- `data/` : données d'exemple
